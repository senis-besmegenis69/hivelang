## Building

Note: the hivelang compiler (hivec) depends on LLVM toolkit.

```shell
cd <hivec-root>
make
```

Optionally, for changing the build configuration to release, or switch to win32, see:

```shell
make help
```

To check overall environment setup for make, see:

```shell
make printvars
```
